Y el cable USB está entre el dedo anular y corazón para ambas manos.
Girar la muñeca en las manecillas del reloj o contrario sigue la misma orientación para ambas manos.
La clase quieto contiene los datos en distintas orientaciones para el shield quieto por su mayor parte.
La clase lento contiene movimientos lentos en distintas orientaciones.
Se tomaron 120 datos con cada mano.
